# Weather-API-App
App implemented with the weather API

## The technologies are used:

  1. for UI: React
  2. for server side: NodeJS

## How to run the project:

##### Install dependencies: 
  ```
  npm i
  ```
##### Run server: 
  
  `npm run server` or `node server`

## Links you may need:
  1. Weather API: https://openweathermap.org/api
